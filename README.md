An interactive Excel Sales Analysis Dashboard designed to analyze sales performance across regions, products, categories, and time.

## Dashboard Highlights
- KPI cards for Total Sales, Profit, Margin %, and Quantity
- Region, Year, and Month slicers for dynamic filtering
- Top 10 products by sales
- Sales analysis by city, region, and segment
- Category-wise quantity and sales contribution
- Daily and monthly sales trend analysis
- One-click Refresh button using VBA
- Full-screen toggle for presentation mode

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- VBA Macros (Refresh & Full Screen Toggle)
- Slicers and dynamic filtering
- Dashboard UI/UX design

## Business Use Case
This dashboard helps stakeholders quickly monitor sales performance, identify top-performing products and regions, and track trends over time for better decision-making.

## How to Use
1. Open the file and enable macros
2. Use slicers to filter data
3. Click the Refresh icon to update data
4. Use the Full Screen icon for presentation view
